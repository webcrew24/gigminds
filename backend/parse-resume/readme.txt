This sample code help you to integrate with RChilli Api for resume parsing.
Sample code can be modified and used as per your need.

Sample code contain 2 file
ParseResumeBinay.py : this file can be used to parse the resume on desktop need to convert to base 64, you also need to set api credential in this file.
ParseResumeUrl.py : this file can be used to parse resume on public url, and you need to set api credential too.
